🛒 Local Store E-commerce Platform

A full-stack e-commerce web application that enables users to browse products, manage purchases, and interact with a seamless online shopping experience.

 Live Demo


GitHub Repository

https://github.com/Hamdallah-alhassan68/locacl-store-E-commerce-Platform

Overview

This project simulates a real-world online store, providing core e-commerce functionalities such as product browsing, cart management, and order tracking. It is designed with scalability and user experience in mind, using modern web technologies.

Features
Product Catalog with images, pricing, and descriptions
Search, Filtering & Sorting for improved product discovery
Shopping Cart for managing selected items
Order Tracking System for monitoring purchases
User Reviews & Feedback for enhanced engagement
Responsive Design for mobile and desktop users
Technologies Used
Frontend: React, HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MongoDB
Deployment: Vercel
APIs: RESTful API architecture
How It Works
User Browsing
Users explore products via categorized listings
Filters and sorting improve search efficiency
Cart Management
Users add/remove items from the shopping cart
Cart dynamically updates in real time
Order Processing
Users place orders through the system
Orders are tracked and updated via backend APIs
User Interaction
Users can leave reviews and interact with products
Impact
Improved product discovery through filtering and sorting features
Streamlined purchasing process with cart and order tracking system
 Enhanced accessibility via responsive cross-device design
System Architecture
Client (React) → REST API (Node.js/Express) → Database (MongoDB)

⚙️ Installation & Setup
# Clone repository
git clone https://github.com/Hamdallah-alhassan68/ecommerce.git

# Navigate into project folder
cd ecommerce

# Install backend dependencies
npm install

# Start backend server
npm start

# Navigate to frontend (if separate)
cd client
npm install
npm start
